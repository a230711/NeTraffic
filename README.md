# NeTraffic - macOS 網路流量監控工具

NeTraffic 是一款專為 macOS 設計的輕量化網路流量監控應用程式，旨在提供即時、直觀的網路數據概覽與詳細統計。

## 🌟 主要功能

- **即時速度監控**：在選單列即時顯示當前的下載與上傳速度。
- **流量圖表**：提供最近 60 秒的流量走勢圖（下載紅色、上傳藍色）。
- **本月流量統計**：
  - **中繼器 (Repeater)**：統計經過中繼器的總流量。
  - **本機 (Mac)**：統計本機電腦的網路使用量。
  - **其他設備**：統計網路中其他連結設備的流量。
- **流量明細查看**：支持查看所有網路供應商的詳細流量分配。
- **智慧自動切換**：在「流量明細」頁面停留超過 1 分半（90 秒）且無操作時，程式會自動跳回「即時概覽」主頁面。
- **標籤自定義**：支持自定義顯示的設備名稱與服務供應商。
- **原生毛玻璃質感**：採用 SwiftUI 與 NSPanel 打造，完美融入 macOS 系統視覺。

## 🚀 安裝與執行

### 環境需求
- macOS 13.0 或以上版本
- 已安裝 Swift 編譯環境

### 編譯與啟動
專案目錄下包含一個 `build_and_run.sh` 腳本，可用於一鍵編譯與重啟：

```bash
./build_and_run.sh
```

## 🛠 專案結構
- `NetworkStatsSrc/`：程式原始碼目錄。
  - `main.swift`：程式入口與選單列邏輯。
  - `ContentView.swift`：UI 介面與自動切換邏輯。
  - `NetworkMonitor.swift`：即時網速監控核心。
  - `StatsJSONReader.swift`：讀取並解析流量統計數據。
- `NeTraffic.app`：產出的應用程式包。
